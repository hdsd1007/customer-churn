# Customer Churn Prediction ML Project

An end-to-end MLOps pipeline for predicting customer churn using machine learning.

## Key Components
- Data processing & feature engineering
- Model training (LightGBM, XGBoost) with hyperparameter tuning
- MLflow experiment tracking
- Model serving via FastAPI/Gradio
- Data validation with Great Expectations
- Docker containerization

## Tech Stack
Python, FastAPI, Gradio, MLflow, scikit-learn, Optuna, Pandas, Docker
